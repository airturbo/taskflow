### Task 3: Create useRouterSync hook

<task id="08-01-T3">
<read_first>
- web/src/hooks/useNavigationState.ts
- web/src/hooks/useFilterState.ts
- web/src/hooks/useViewConfig.ts

<action>
Create web/src/hooks/useRouterSync.ts.
This hook:
1. Parses the current URL hash path + query params into state values
2. Provides a `navigateTo(selection, pushHistory)` function that updates URL
3. Provides `syncViewState(params)` that updates query params (replaceState)
4. Listens to location changes (useLocation) and calls provided setters

URL structure:
- Path: /, /focus, /all, /upcoming, /inbox, /logbook, /trash, /list/:id, /filter/:id
- Query: view, q, cal, anchor, scale, tags, task, cal_done

Parse functions:
- pathToSelection(pathname): string → activeSelection value
- selectionToPath(activeSelection): string → URL path
- parseQueryParams(search): parsed object
- buildQueryString(params): search string
</action>
<acceptance_criteria>
- Hook exports: useRouterSync
- Returns: { navigateTo, syncViewState, parsedState }
- All URL formats parse correctly
- Build passes
</acceptance_criteria>
</task>
