# Plan 09-04: 卡片分类理由 pill + hover tooltip

## 目标

在 MatrixView 卡片上显示分类理由 badge（pill），hover 时展示 tooltip 解释分类依据。

## Tasks

<task id="09-04-T1">
  <title>MatrixView — 卡片添加 quadrant badge pill</title>
  <read_first>
    - web/src/components/views/MatrixView.tsx
    - web/src/components/views/MatrixView.module.css
  </read_first>
  <action>
    在矩阵卡片的底部标签区域（chip-wrap dense）之前，新增一个 quadrant badge：
    
    ```tsx
    <span 
      className={styles.matrixQuadrantBadge}
      data-quadrant={key}
      title={getQuadrantReasonTooltip(task)}
    >
      {getQuadrantShortLabel(key)}
    </span>
    ```
    
    颜色方案：
    - Q1（紧急+重要）: 红色
    - Q2（重要）: 蓝色  
    - Q3（紧急）: 橙色
    - Q4（普通）: 灰色（只在 q4 不显示，或显示为低调样式）
    
    新增辅助函数 `getQuadrantReasonTooltip(task)`：
    - Q1: "紧急且重要"
    - Q2: "重要，不紧急"
    - Q3: "紧急，不重要"
    - Q4: "待分类"
    
    在 MatrixView.module.css 中添加 `.matrixQuadrantBadge` 样式。
  </action>
  <acceptance_criteria>
    - 每张卡片显示小 badge
    - hover 显示 tooltip（使用 HTML title 属性）
    - Q4 badge 显示为低调样式（灰色）
    - 视觉上与现有 chip-wrap 协调
    - TypeScript 编译通过
  </acceptance_criteria>
</task>
