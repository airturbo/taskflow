# 05-PLAN-02: Mobile Completion — Confirm Dialog to Undo Toast

## Summary

Replace the blocking MobileConfirmSheet in the task completion flow with an immediate toggle + Undo Toast (3s auto-dismiss). The confirm dialog remains available for destructive operations (delete folder/list) — only the completion path changes.

Current flow: tap check → MobileConfirmSheet "确认完成？" → confirm → toggle + toast
Target flow: tap check → immediate toggle → show Undo Toast (3s) → auto-dismiss

The completion Toast infrastructure already exists in `mobileUiStore` (`completionToast`, `showCompletionToast`, `hideCompletionToast`) and the Toast UI is already rendered in App.tsx (line 3359-3391).
