<task id="02" title="Offline queue 500-cap + 7-day expiry">
<read_first>
- web/src/utils/offline-queue.ts
</read_first>
<action>
1. In `web/src/utils/offline-queue.ts`, add constants after `const QUEUE_KEY`:
   ```ts
   const MAX_QUEUE_SIZE = 500
   const MAX_AGE_MS = 7 * 24 * 60 * 60 * 1000 // 7 days
   ```

2. Add `enqueuedAt` field tracking to QueueEntry (already exists as string).

3. In `enqueueOfflineState`, after merging tasks, add capacity enforcement:
   - After computing `mergedTasks`, sort by `updatedAt` descending
   - If `mergedTasks.length > MAX_QUEUE_SIZE`, slice to keep only the newest 500:
     ```ts
     // Capacity enforcement: keep newest MAX_QUEUE_SIZE tasks
     if (mergedTasks.length > MAX_QUEUE_SIZE) {
       mergedTasks.sort((a, b) => (b.updatedAt ?? '').localeCompare(a.updatedAt ?? ''))
       mergedTasks.length = MAX_QUEUE_SIZE
     }
     ```

4. In `flushOfflineQueue`, before the upsert call, add expiry check:
   ```ts
   // Expiry check: discard queue if too old
   const ageMs = Date.now() - new Date(entry.enqueuedAt).getTime()
   if (ageMs > MAX_AGE_MS) {
     window.localStorage.removeItem(QUEUE_KEY)
     return true // expired, treat as flushed
   }
   ```

5. Add a new exported function for external queue health check:
   ```ts
   /** Get queue metadata for diagnostics */
   export const getQueueStats = (): { size: number; ageMs: number } | null => {
     if (typeof window === 'undefined') return null
     const raw = window.localStorage.getItem(QUEUE_KEY)
     if (!raw) return null
     try {
       const entry = JSON.parse(raw) as QueueEntry
       return {
         size: entry.state.tasks?.length ?? 0,
         ageMs: Date.now() - new Date(entry.enqueuedAt).getTime(),
       }
     } catch { return null }
   }
   ```
</action>
