# Plan 04 — DATE-04: Calendar view dual-day marking

## Objective
在日历视图月历格子中，为同时有 dueAt 和 deadlineAt 的任务在两个日期都显示标记：
- dueAt 日期格：蓝色圆点
- deadlineAt 日期格：红色圆点（已有 TaskDeadlineDot，但只在任务卡片上，不在日期格头部）

实现方案：在月历每个日期格的 header 下方，统计该日 dueAt 任务数量（蓝点）和 deadlineAt 任务数量（红点），显示为小指示点行。

## Tasks

<task id="04-1">
In CalendarView.tsx month mode, compute per-day dueAt count and deadlineAt count.
Show a row of colored dots in each day cell header indicating dual-date presence.
</task>

<task id="04-2">
Add CSS in CalendarView.module.css for the dual-date dot indicators.
</task>
