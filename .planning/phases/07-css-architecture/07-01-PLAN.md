<task id="A2">
<title>Implement MINOR-04: thin auto-hide scrollbar for .main-stage</title>
<read_first>
- web/src/styles/globals.css (just created in A1)
- web/src/index.css (L126–L172: Layout section containing .main-stage at L151)
- .planning/phases/07-css-architecture/07-RESEARCH.md (Section 5: MINOR-04 details)
</read_first>
<action>
Append to `web/src/styles/globals.css` a new section for `.main-stage` scrollbar:

```css
/* ---- MINOR-04: Main content area thin auto-hide scrollbar ---- */
.main-stage {
  scrollbar-width: thin;
  scrollbar-color: rgba(128,128,128,0.2) transparent;
}
.main-stage::-webkit-scrollbar {
  display: block;
  width: 4px;
  height: 4px;
}
.main-stage::-webkit-scrollbar-track {
  background: transparent;
}
.main-stage::-webkit-scrollbar-thumb {
  background: rgba(128,128,128,0);
  border-radius: 999px;
  transition: background 0.3s ease;
}
.main-stage:hover::-webkit-scrollbar-thumb {
  background: rgba(128,128,128,0.2);
}
.main-stage:hover::-webkit-scrollbar-thumb:hover {
  background: rgba(128,128,128,0.4);
}
```

This overrides the global `scrollbar-width: none` for `.main-stage` specifically, giving it a thin scrollbar that shows on hover (auto-hide effect via opacity).
</action>

- globals.css contains `.main-stage:hover::-webkit-scrollbar-thumb {` with `rgba(128,128,128,0.2)`

</acceptance_criteria>
