# Plan C: View CSS Modules Extraction

## Goal
Extract the 6 view-specific CSS sections from index.css into individual `.module.css` files under `src/components/views/`. Each view is self-contained, making this a clean extraction.

## Dependency Note
Plan C depends on Plan B (not just Plan A) to avoid merge conflicts on `index.css`. Plan B removes desktop component sections first; Plan C then removes view sections from the already-reduced index.css. Plan C owns **only** L1423–L2362 (view sections) — all other ranges are handled by Plan A or Plan B.
