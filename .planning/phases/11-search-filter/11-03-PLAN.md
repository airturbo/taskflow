# Plan 11-03: NLP 解析预览 chip

## Objective
在 CommandPalette 输入框下方，实时解析用户输入，
显示结构化 filter chip（日期/标签/优先级/状态），
并提供 "Apply to workspace" 功能。

## NLP Parsing Rules
| Input Pattern | Parsed |
|--------------|--------|
| `#tagname` | tag chip (match existing tags by name) |
| `!urgent/!高/!high/!p1` | priority chip |
| `status:todo/doing/done` | status chip |
| `due:today/week/overdue` | due chip |
| `@listname` | list chip (show in UI, not applied to FilterState) |
| remaining text | keyword chip |

## UI Design
- chip row: flex wrap, under input, above results
- chip: rounded pill with icon + label + × remove button
- "Apply filters" button: appears when any filter chip present, bottom of chip row
- Cmd+Enter shortcut: apply + close palette

## Chip Styles
- tag chip: colored dot + tag name (tag.color background tint)
- priority chip: priority color (urgent=red, high=amber, normal=purple, low=slate)
- status chip: neutral
- due chip: warning color for overdue
- keyword chip: neutral with search icon
