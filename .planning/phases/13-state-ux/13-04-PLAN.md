# Plan 04 — UX-05: InlineCreatePopover @floating-ui/react Positioning

## Goal
用 @floating-ui/react 替换 InlineCreatePopover 中的手写边界约束逻辑。
保留自由拖拽行为，用 floating-ui 的 shift/flip middleware 做边界 clamping。

## Approach
InlineCreatePopover 是自由浮动（非 anchor-relative），所以 @floating-ui/react 的
useFloating hook 用 "virtual reference element" 模式：将当前拖拽位置作为虚拟 anchor。

实际上更合适的是用 `@floating-ui/dom` 的 `computePosition` + `shift` middleware
在 handlePointerUp 时做边界校正，替换手写的 `clampInlineCreatePosition`。
