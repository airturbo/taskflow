# Plan 03 — UX-04: Zustand Consolidate UI State

## Goal
将分散在 useState hooks 中的 UI 状态迁移到 Zustand stores：
1. `useModalState` → `desktopUiStore` (已在 Plan 01 中创建)
2. App.tsx 中的 mobile extra states → `mobileUiStore` 扩展
3. App.tsx 直接使用 stores，不再通过 hook 桥接

## Strategy
WorkspaceShell props 不变（避免大规模重构），只把 App.tsx 的 useState 来源替换为 stores。
