# Plan 01 — UX-02: Desktop Completion Animation + Undo Toast

## Goal
当用户在桌面端勾选完成任务时，展示：
1. ✓ 图标弹跳动画（bounce）
2. 任务标题 strikethrough 过渡
3. 任务卡片 slide-out（延迟 ~900ms 后滑出消失）
4. 顶部 Undo Toast 出现（可点击撤销，4s 自动消失）

## Tasks

- @keyframes task-slide-out: translateX(-24px) + opacity 0 + max-height 0
- .check-button.is-completing: apply bounce animation
- .task-card.is-completing h3: text-decoration: line-through transition
- .task-card.is-completing: apply slide-out animation (delay 800ms)
</task>
