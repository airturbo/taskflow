# Plan 15-03: IndexedDB 离线队列升级

## Objective
将离线队列从 localStorage 迁移至 IndexedDB，并添加指数退避重试逻辑。

## Tasks

<task id="15-03-A">
### A: Install idb library
- `cd web && npm install idb`
- idb: ~1.3KB gzipped, TypeScript-native IndexedDB wrapper
</task>

<task id="15-03-B">
### B: Rewrite offline-queue.ts with IndexedDB
- DB: `taskflow-offline-db`, store: `offline-queue`, keyPath: `userId`
- QueueEntry gains: `retryCount: number`, `nextRetryAt: string | null`
- Exponential backoff: base=2s, factor=2, max=5min (300s)
- `enqueueOfflineState`: open DB, get existing entry, merge tasks, upsert
- `flushOfflineQueue`: check nextRetryAt, if not ready skip; on fail increment retryCount + set nextRetryAt
- `hasPendingQueue`: check IDB store count
- `clearOfflineQueue`: delete record from IDB
- Keep legacy localStorage migration: on first load, if localStorage has queue data, import to IDB and delete LS entry
- All functions remain async (already were)
</task>

<task id="15-03-C">
### C: Update callers in App.tsx
- `enqueueOfflineState` is already called; no signature change needed
- `flushOfflineQueue` already called on online event — no change needed
- `hasPendingQueue` → make sure caller handles Promise<boolean>
</task>

<task id="15-03-D">
### D: Commit
