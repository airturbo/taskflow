## 📋 执行摘要

TaskFlow 是一个采用 **Tauri + React 19 + TypeScript** 构建的全端任务管理应用，具有业界领先的**响应式设计**和**触摸交互**支持。项目采用了**渐进式响应式设计**（Responsive-First），在单一代码库中同时支持桌面端（1480x960）和移动端（≤680px）。

### 核心特性
✅ **三层响应式断点系统** - 基于 CSS Media Query  
✅ **iOS 风格底部 Sheet** - 手势识别 + 动画  
✅ **5 种内容视图** + 移动端 Tab 导航  
✅ **Pointer Events 拖拽系统** - 日历/看板/时间线  
✅ **Touch 手势识别** - 下滑关闭、滚动检测  
✅ **自然语言任务解析** - 智能提取时间/优先级/标签

### 5.1 Task Sheet 下滑关闭手势（行 6196-6272）

这是 iOS 风格的交互实现，**最精彩的部分**！

#### 手势检测逻辑

```typescript
const gestureRef = useRef<{
  startY: number
  startTime: number
  currentY: number
  isDragging: boolean
  source: 'handle' | 'body'  // 从把手还是内容区开始
} | null>(null)

const DISMISS_DISTANCE = 72      // 下拉 72px 即触发关闭
const DISMISS_VELOCITY = 0.5     // 或速度 > 0.5 px/ms 即关闭

// ---- 把手区域（始终可拖动） ----
const handleHandleTouchStart = (e: React.TouchEvent) => {
  gestureRef.current = {
    startY: e.touches[0].clientY,
    startTime: Date.now(),
    currentY: e.touches[0].clientY,
    isDragging: true,
    source: 'handle',
  }
}

// ---- 内容区域（仅在顶部可拖动） ----
const handleBodyTouchStart = (e: React.TouchEvent) => {
  const bodyEl = bodyRef.current
  const isAtTop = !bodyEl || bodyEl.scrollTop <= 0  // 关键：检查滚动位置
  gestureRef.current = {
    startY: e.touches[0].clientY,
    startTime: Date.now(),
    currentY: e.touches[0].clientY,
    isDragging: false,  // 先不拖动，等待触摸移动
    source: 'body',
  }
  if (isAtTop) {
    // 即便在顶部也不立即启动，等待下拉确实发生
  }
}

const handleTouchMove = (e: React.TouchEvent) => {
  const g = gestureRef.current
  if (!g) return
  const clientY = e.touches[0].clientY
  g.currentY = clientY
  const delta = clientY - g.startY

  if (g.source === 'handle') {
    // 把手区域：总是拖动
    if (delta > 0) {
      e.preventDefault()
      applySheetTransform(delta)
    }
  } else if (g.source === 'body') {
    const bodyEl = bodyRef.current
    const isAtTop = !bodyEl || bodyEl.scrollTop <= 0
    
    if (!g.isDragging && isAtTop && delta > 8) {
      // 从顶部开始明显下拉 > 8px
      g.isDragging = true
      g.startY = clientY        // 重置起点（平滑过渡）
      g.startTime = Date.now()
    }
    
    if (g.isDragging && delta > 0) {
      e.preventDefault()
      applySheetTransform(clientY - g.startY)
    }
  }
}

const handleTouchEnd = () => {
  const g = gestureRef.current
  if (!g) return
  const delta = g.currentY - g.startY
  const elapsed = Date.now() - g.startTime
  const velocity = elapsed > 0 ? delta / elapsed : 0

  if (g.isDragging || g.source === 'handle') {
    if (delta > DISMISS_DISTANCE || velocity > DISMISS_VELOCITY) {
      // 触发关闭动画
      if (sheetRef.current) {
        sheetRef.current.style.transition = 'transform 0.25s cubic-bezier(0.32, 0.72, 0, 1)'
        sheetRef.current.style.transform = 'translateY(100%)'
      }
      window.setTimeout(onClose, 250)
    } else {
      // 弹回到原位
      resetSheetTransform()
    }
  }
  gestureRef.current = null
}
```

#### 关键设计点

| 特性 | 实现 |
|-----|------|
| **两个触摸区域** | handle（把手）总是可拖 / body（内容）仅顶部可拖 |
| **滚动检测** | `bodyEl.scrollTop <= 0` 防止与内容滚动冲突 |
| **动态启动** | `delta > 8px` 才启动拖动，避免误触 |
| **物理动画** | velocity 计算 + cubic-bezier 缓动 |
| **起点重置** | 进入拖动时重置 `startY` 和 `startTime` |
| **平滑回弹** | CSS transition 实现动画 |

```typescript
const finalizePointerDrag = (event: PointerEvent) => {
  const current = pointerDragRef.current
  if (!current || event.pointerId !== current.pointerId) return

  const targetDateKey = current.dragged 
    ? dragOverDateKeyRef.current ?? resolveDropDateKey(event.clientX, event.clientY) 
    : null
    
  if (current.dragged && targetDateKey) {
    onMoveTaskToDate(current.taskId, current.sourceDateKey, targetDateKey)
    setFocusedDateKey(targetDateKey)
    markClickSuppressed(suppressClickRef)  // 防止 click 触发
  }

  finishDrag()  // 重置所有拖动状态
}
```
