## 1. 应用启动流程

```
[打开应用]
    │
    ├── 首次访问？
    │   ├── 是 → [欢迎引导页]
    │   │        │
    │   │        ├── Step 1: 选择使用场景（个人/工作/学生）
    │   │        ├── Step 2: 选择主题偏好（亮/暗）
    │   │        ├── Step 3: 导入数据？（滴答清单/Todoist/跳过）
    │   │        └── Step 4: 创建第一个任务（可跳过）
    │   │               │
    │   │               └── → [今日视图]
    │   │
    │   └── 否 → 检查登录状态
    │            │
    │            ├── 已登录 → [同步云端数据] → [今日视图]
    │            │
    │            └── 未登录 → [读取本地 IndexedDB] → [今日视图]
    │
    └── → [今日视图 = 默认首页]
```

## 2. 任务创建流程

### 2.1 快速创建

```
[任意页面]
    │
    ├── 触发方式:
    │   ├── 点击顶栏/底部 [+] 按钮
    │   ├── 按 Ctrl/Cmd + N
    │   └── 在列表底部输入框直接输入
    │
    └── → [快速创建面板] (浮层)
              │
              ├── 输入标题: "明天下午3点和张三开会 #工作 !紧急"
              │       │
              │       └── [智能解析引擎]
              │               │
              │               ├── 识别日期: "明天下午3点" → 2026-03-31 15:00
              │               ├── 识别标签: "#工作"
              │               ├── 识别优先级: "!紧急" → 🔴紧急
              │               └── 识别人名: "张三" → 提示指派
              │
              ├── 可选操作:
              │   ├── 选择清单 (默认收件箱)
              │   ├── 设置优先级
              │   ├── 选择日期
              │   └── 添加标签
              │
              ├── 按 Enter → [创建成功]
              │       │
              │       ├── 连续创建模式关闭 → 面板关闭, Toast "任务已创建"
              │       └── 连续创建模式开启 → 面板清空, 继续输入下一条
              │
              └── 按 Esc → 取消, 面板关闭
```

### 2.2 日历拖拽创建

```
[日历视图 (周视图/日视图)]
    │
    └── 在空白时间段拖拽
            │
            ├── [创建占位块] 跟随鼠标
            │   显示预估时间范围
            │
            └── 松手 → [弹出快速创建面板]
                        │
                        ├── 日期和时间已预填
                        ├── 输入标题
                        └── 确认创建
```

## 3. 任务管理流程

### 3.1 完成任务

```
[任务列表中点击复选框]
    │
    ├── [✓ 打勾动画] (200ms, ease-spring)
    │
    ├── [文字划线效果] (200ms)
    │
    ├── [延迟淡出] (1000ms 后从列表移除)
    │       │
    │       └── 在淡出期间可点击撤销
    │
    ├── [底部 Toast] "任务已完成" + [撤销] 按钮 (5秒)
    │       │
    │       ├── 点击撤销 → 任务恢复到原位
    │       └── 5秒后 Toast 消失, 任务移入"已完成"
    │
    └── 如果是重复任务:
            │
            └── 自动创建下一个周期的任务实例
```

### 3.2 删除任务

```
[触发删除]
    │
    ├── 方式: 右键菜单 "删除" / 选中后按 Delete / 滑动删除(移动端)
    │
    ├── [任务从列表淡出]
    │
    ├── [底部 Toast] "任务已删除" + [撤销] 按钮 (5秒)
    │       │
    │       ├── 点击撤销 → 恢复到原位
    │       └── 超时 → 任务进入回收站
    │
    └── 回收站规则: 30天后自动永久删除
```

### 3.3 批量操作

```
[进入多选模式]
    │
    ├── 触发: Ctrl/Cmd + 点击第一个任务, 或长按(移动端)
    │
    ├── [顶部操作栏替换为批量工具栏]
    │   "已选 3 项"  [🚩优先级] [📂移动] [🏷标签] [🗑删除] [✕取消]
    │
    ├── 继续 Ctrl+点击 或 Shift+点击范围选
    │
    └── 选择操作 → 应用到所有选中任务 → 退出多选模式
```

## 5. 番茄专注流程

```
[进入番茄专注]
    │
    ├── 入口1: 侧边栏 "番茄专注"
    ├── 入口2: 任务右键 "开始番茄"
    └── 入口3: 快捷键 Ctrl+Shift+F
    │
    └── [番茄专注页面]
            │
            ├── [选择关联任务] (可选)
            │
            ├── [选择白噪音] (可选)
            │   ├── 选择音效类型
            │   ├── 调节各音效音量
            │   └── 音效混合预览
            │
            ├── [开始专注] →→→ [倒计时 25分钟]
            │                       │
            │                       ├── 运行中可:
            │                       │   ├── 暂停 → 恢复
            │                       │   ├── 放弃 → 确认弹窗 → 记录为未完成
            │                       │   └── 严格模式下: 不可暂停/放弃
            │                       │
            │                       └── 倒计时结束
            │                               │
            │                               ├── [声音+通知提醒]
            │                               ├── [记录番茄到任务]
            │                               │
            │                               ├── 判断: 是否第4个番茄?
            │                               │   ├── 否 → [短休息 5分钟]
            │                               │   └── 是 → [长休息 15分钟]
            │                               │
            │                               └── 休息结束 → [开始下一个番茄?]
            │                                       ├── 是 → 回到倒计时
            │                                       └── 否 → 结束专注
            │
            └── [查看今日专注统计]
```

## 6. 习惯打卡流程

```
[进入习惯打卡]
    │
    ├── [今日习惯列表]
    │       │
    │       ├── 简单打卡类:
    │       │   └── 点击 [✓] → 打卡成功动画 → 连续天数+1
    │       │
    │       ├── 计数打卡类 (如喝水8杯):
    │       │   └── 点击 [+1] → 进度条更新 → 达标时庆祝动画
    │       │
    │       └── 时长打卡类 (如运动30分钟):
    │           └── [开始计时] → 倒计时/正计时 → [完成] → 记录时长
    │
    ├── 打卡时可选:
    │   └── [添加日志] → 输入文字备注 → 保存
    │
    └── [查看统计]
            ├── 月度热力图
            ├── 连续天数
            └── 完成率趋势
```

## 7. 协作流程

```
[分享清单]
    │
    ├── 点击清单右键 → "分享"
    │
    ├── [分享弹窗]
    │   ├── 输入用户邮箱/ID
    │   ├── 设置权限: 可编辑 / 仅查看
    │   ├── 生成分享链接
    │   └── [发送邀请]
    │
    └── 对方接受后:
            │
            ├── 清单出现在对方侧边栏 "共享" 分组下
            │
            ├── [实时协作]:
            │   ├── 任何人创建/编辑/完成任务 → 其他人实时看到变更
            │   ├── 指派任务给成员 → 被指派人收到通知
            │   └── 评论任务 → 相关人收到通知
            │
            └── [动态面板]:
                    └── 显示所有成员的操作历史
```

## 8. 搜索与命令面板

```
[按 Ctrl/Cmd + K]
    │
    └── [命令面板打开] (居中浮层, 暗色遮罩)
            │
            ├── 输入框聚焦
            │
            ├── 空状态 → 显示最近操作 + 快捷命令
            │
            ├── 输入文字:
            │   │
            │   ├── 匹配任务标题/描述/标签 → 显示任务搜索结果
            │   │   └── 点击任务 → 跳转到该任务并展开详情
            │   │
            │   └── 匹配命令 → 显示命令列表
            │       └── 选择命令 → 执行 (新建任务/切换视图/打开设置...)
            │
            └── 按 Esc → 关闭面板
```
