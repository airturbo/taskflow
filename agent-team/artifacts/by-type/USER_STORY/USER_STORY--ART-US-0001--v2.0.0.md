## 一、P0 用户故事（MVP 核心）

---

### US-001 快速创建任务

> **作为**用户，**我希望**用最少的操作快速创建一条待办，**以便**不遗漏任何突然想到的事情。

**验收标准：**
- [ ] AC-001-1：点击 "+" 按钮或快捷键 Ctrl/Cmd+N 弹出创建面板，光标自动聚焦标题
- [ ] AC-001-2：输入标题后按回车即创建成功，待办出现在列表中
- [ ] AC-001-3：空标题不允许提交，给出错误提示
- [ ] AC-001-4：创建时可选填：清单、优先级、截止日期、标签
- [ ] AC-001-5：未指定清单的任务自动归入"收件箱"
- [ ] AC-001-6：支持连续创建模式（创建后面板不关闭，可继续添加下一条）
- [ ] AC-001-7：智能日期识别——输入"明天下午3点开会"自动解析为截止日期和提醒
- [ ] AC-001-8：整个创建流程 ≤ 3 秒

> **作为**用户，**我希望**通过完成任务获得成就和积分，**以便**获得持续的激励感。

### US-202 桌面便签

> **作为**用户，**我希望**将待办以便签形式固定在桌面，**以便**随时可见不遗忘。

### US-203 多端云同步

> **作为**用户，**我希望**在不同设备间实时同步数据，**以便**在任何设备上都能使用。

### US-204 PWA 离线使用

> **作为**用户，**我希望**在断网时也能正常使用，**以便**不受网络条件影响。
