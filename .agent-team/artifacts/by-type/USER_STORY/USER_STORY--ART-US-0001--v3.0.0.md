## P0 用户故事

### US-301 快速录入
作为高频记录任务的用户，我希望能直接在顶部输入“明天下午3点开会”并创建任务，这样我不用跳出当前上下文。
- AC-301-1：支持 Enter 直接创建
- AC-301-2：至少识别今天/明天/后天/几点等基础日期词
- AC-301-3：创建后任务自动进入详情面板

### US-302 高级任务详情
作为重度任务管理用户，我希望在详情面板里维护子任务、提醒、重复、标签和预计番茄，这样一个任务就能承载完整上下文。
- AC-302-1：详情面板可编辑核心属性
- AC-302-2：支持增删子任务
- AC-302-3：支持多个提醒项展示

### US-303 系统清单与自定义清单
作为需要管理多个领域事项的用户，我希望同时拥有今日、最近7天、收件箱、已完成以及自定义清单，这样我能在不同层级间切换。
- AC-303-1：系统清单可见并可切换
- AC-303-2：支持文件夹/清单层级展示

### US-304 标签与过滤器
作为需要按场景聚焦的用户，我希望通过标签和智能清单快速筛出相关任务，这样我能减少信息噪音。
- AC-304-1：支持标签展示与点击过滤
- AC-304-2：支持至少两个预置智能清单

### US-305 多视图切换
作为不同规划方式的用户，我希望在列表、日历、看板、时间线、四象限之间切换，这样我能按问题选择视角。
- AC-305-1：至少四个视图真实可用
- AC-305-2：切换视图不丢失当前筛选上下文

### US-306 番茄专注
作为执行任务的用户，我希望把番茄专注关联到某个任务并累计投入时长，这样我可以把计划与执行连接起来。
- AC-306-1：支持开始/暂停/重置
- AC-306-2：支持关联任务
- AC-306-3：完成专注后任务时长更新

### US-307 习惯打卡
作为长期养成习惯的用户，我希望记录日常习惯并看到连续天数和完成进度，这样我能形成反馈闭环。
- AC-307-1：支持至少三种习惯记录形态之一的组合展示
- AC-307-2：完成后今日进度变化可见

### US-308 统计面板
作为复盘型用户，我希望看到任务、专注和习惯的统计概览，这样我知道自己是否真的在推进事情。
- AC-308-1：统计页可见核心指标
- AC-308-2：指标随操作变化而变化

## P1 用户故事

### US-309 协作形态
作为团队协作用户，我希望任务里能看到指派、评论和活动流，即使本轮不做真实多人同步，也能先验证协作信息结构。

### US-310 数据持续可用
作为长期使用者，我希望刷新页面后数据不丢失，这样 Demo 不会像一次性展示页。

### US-311 已完成与回收站
作为谨慎操作的用户，我希望已完成和已删除任务都能被找回，这样我不用担心误操作。

### US-312 高级能力默认开放
作为不想被会员墙打断的用户，我希望日历、过滤器、统计、预计番茄这些高级能力默认可见可用。
