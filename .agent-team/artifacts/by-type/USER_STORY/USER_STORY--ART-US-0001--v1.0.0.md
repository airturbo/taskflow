## P0 用户故事（MVP 必须）

### US-001 新建待办

> **作为**一个用户，**我希望**能快速创建一条新的待办事项，**以便**我不会遗忘突然想到的任务。

**验收标准：**
- [ ] AC-001-1：点击 "+" 按钮后，弹出创建面板，光标自动聚焦标题输入框
- [ ] AC-001-2：输入标题后按回车或点击"创建"按钮，待办成功创建并出现在列表中
- [ ] AC-001-3：标题为必填项，空标题不允许提交，给出提示
- [ ] AC-001-4：创建时可选填清单归属、优先级、截止日期
- [ ] AC-001-5：未指定清单的待办归入"收件箱"
- [ ] AC-001-6：整个创建流程 ≤ 3 秒完成

---

### US-002 编辑待办

> **作为**一个用户，**我希望**能修改已创建的待办内容，**以便**随时更新任务的详细信息。

**验收标准：**
- [ ] AC-002-1：点击待办后展开详情面板
- [ ] AC-002-2：可修改标题、描述、优先级、截止日期、所属清单
- [ ] AC-002-3：修改后自动保存，无需手动点击保存按钮
- [ ] AC-002-4：标题不能被修改为空

---

### US-003 完成待办

> **作为**一个用户，**我希望**一键标记任务为已完成，**以便**我能清楚知道哪些事已经做完了。

**验收标准：**
- [ ] AC-003-1：点击待办前的圆形复选框，待办被标记为已完成
- [ ] AC-003-2：完成时有视觉动画反馈（如打勾动画 + 文字划线）
- [ ] AC-003-3：已完成的待办延迟约 1 秒后从当前列表淡出
- [ ] AC-003-4：已完成的待办可在"已完成"视图中查看
- [ ] AC-003-5：可以取消完成（恢复为未完成状态）

---

### US-004 删除待办

> **作为**一个用户，**我希望**能删除不再需要的待办，**以便**保持列表清爽。

**验收标准：**
- [ ] AC-004-1：支持通过滑动或右键菜单触发删除
- [ ] AC-004-2：删除后底部弹出"已删除"提示条，含"撤销"按钮（5 秒内可撤销）
- [ ] AC-004-3：删除的待办进入回收站，不立即永久删除
- [ ] AC-004-4：回收站中的待办 30 天后自动永久清理

---

### US-005 清单管理

> **作为**一个用户，**我希望**能创建不同的清单来分类管理待办，**以便**工作和生活的任务不会混在一起。

**验收标准：**
- [ ] AC-005-1：支持创建自定义清单，输入名称即可创建
- [ ] AC-005-2：支持为清单选择图标或颜色
- [ ] AC-005-3：可将待办拖拽或手动移动到不同清单
- [ ] AC-005-4：支持编辑清单名称
- [ ] AC-005-5：删除清单时，其中的待办转移到"收件箱"
- [ ] AC-005-6：系统默认清单（收件箱、今日、已完成）不可删除

---

### US-006 优先级设置

> **作为**一个用户，**我希望**能为待办设置优先级，**以便**我能区分哪些事最紧急需要先做。

**验收标准：**
- [ ] AC-006-1：提供四级优先级：🔴 紧急 / 🟡 高 / 🔵 普通 / ⚪ 低
- [ ] AC-006-2：新建待办默认优先级为"普通"
- [ ] AC-006-3：优先级以颜色标识，在列表中一眼可辨
- [ ] AC-006-4：支持按优先级筛选和排序

---

### US-007 截止日期设置

> **作为**一个用户，**我希望**能为待办设置截止日期，**以便**我能把控时间节点不至于超期。

**验收标准：**
- [ ] AC-007-1：点击日期区域弹出日期选择器
- [ ] AC-007-2：支持快捷选项：今天、明天、下周一、自定义日期
- [ ] AC-007-3：已过期的待办以红色文字标注过期天数
- [ ] AC-007-4：今日到期的待办在"今日"视图中展示

---

### US-008 今日视图

> **作为**一个用户，**我希望**有一个"今日"专属视图，**以便**每天聚焦当天最重要的事。

**验收标准：**
- [ ] AC-008-1：今日视图展示截止日期为今天的待办
- [ ] AC-008-2：同时展示已过期但未完成的待办（高亮标红）
- [ ] AC-008-3：应用启动后默认进入今日视图
- [ ] AC-008-4：今日视图中的待办按优先级降序排列

## P1 用户故事（应该有）

### US-101 标签管理

> **作为**一个用户，**我希望**能给待办打标签，**以便**用另一个维度来组织和检索任务。

**验收标准：**
- [ ] AC-101-1：支持创建自定义标签，可设置名称和颜色
- [ ] AC-101-2：一个待办可以添加多个标签
- [ ] AC-101-3：支持按标签筛选待办
- [ ] AC-101-4：在列表中以小色块展示待办的标签

---

### US-102 全局搜索

> **作为**一个用户，**我希望**能搜索所有待办，**以便**在待办很多时快速找到目标任务。

**验收标准：**
- [ ] AC-102-1：顶部提供搜索框，支持快捷键（Ctrl/Cmd + K）唤起
- [ ] AC-102-2：按标题和描述进行模糊匹配
- [ ] AC-102-3：搜索结果实时展示（输入即搜索）
- [ ] AC-102-4：搜索结果高亮匹配关键词

---

### US-103 到期提醒

> **作为**一个用户，**我希望**在任务到期前收到提醒，**以便**不会错过重要的截止日期。

**验收标准：**
- [ ] AC-103-1：到期当天上午 9:00 推送浏览器通知
- [ ] AC-103-2：用户可自定义提前提醒时间（1小时/3小时/1天）
- [ ] AC-103-3：若浏览器通知被拒绝，则应用内显示红色徽标
- [ ] AC-103-4：提醒通知点击后跳转到对应待办详情

---

### US-104 拖拽排序

> **作为**一个用户，**我希望**能拖拽调整待办的顺序，**以便**按自己的节奏安排任务的执行先后。

**验收标准：**
- [ ] AC-104-1：长按/拖拽待办项可改变其在列表中的位置
- [ ] AC-104-2：拖拽时有视觉反馈（阴影、占位符）
- [ ] AC-104-3：排序结果即时持久化

---

### US-105 子任务

> **作为**一个用户，**我希望**能为一个待办添加子任务，**以便**将大任务拆解为可执行的小步骤。

**验收标准：**
- [ ] AC-105-1：在待办详情中可添加子任务（checklist 形式）
- [ ] AC-105-2：子任务可独立勾选完成
- [ ] AC-105-3：父任务显示子任务完成进度（如 2/5）
- [ ] AC-105-4：所有子任务完成后提示是否将父任务也标记为完成
